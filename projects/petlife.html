Accueil
  • Carte propriétaire avec photo (persistante entre sessions)
  • Badge urgences sur chaque animal :
    ‣ Rouge si vaccin obligatoire manquant
    ‣ Ambre si soins en retard
  • Compteur total urgences avec wording adapté ("1 vaccin obligatoire" vs "X soins à faire")
  • Accès rapide :
    ‣ 📅 PilePoils → popup App / Site web
    ‣ 💊 Rappels → popup animaux avec soins en retard, badge rouge/ambre selon gravité
    ‣ 🛡️ Assurance → popup boutons App 📱 / Site 🌐 pour chaque animal
    ‣ 🚨 Urgences → écran dédié avec contacts géolocalisés
  • Suppression / édition animal par swipe

  • Date prochaine administration automatique selon le protocole
  • Arrêter / Reprendre un traitement
  • Confirmation traitement du jour (✓ / ✗)
  • Valeurs de référence vitales par espèce
  • Guide couleurs des muqueuses
  • Encyclopédie maladies par espèce

Profil animal
  • Création en 3 étapes (Infos générales / Santé / Vétérinaire)

  • Antécédents médicaux
  • Option décès (date + heure facultative)
